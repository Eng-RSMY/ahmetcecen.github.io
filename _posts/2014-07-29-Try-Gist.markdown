Using the Sherman-Morrison formula:

{% highlight matlab %}
function CV=LOOCVLinRegress(X,y)
[Q,R] = qr(X,0);
H=Q*Q';yhat=X*R\(Q'*R);
CV=(y-yhat)./(1-diag(H));
{% endhighlight %}
